# deno-sample-proj

**Deno** - Simple, Modern, and Secured

- Package manager is decentralized
- Single executable to get Deno up and running.
  `deno --help` - view the help commands
  `deno upgrade` - install latest version of deno
  `deno fmt` - format your code

- You can add completion to deno

## Import Module

- Reference module whereever they live on the internet

## Opt-in to run Web Server

- The order is important.

```bash
deno run --allow-net webserver.ts
```

- [Deno Server](https://deno.land/std@0.83.0/http)
